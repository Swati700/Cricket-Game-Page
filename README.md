## 🏏 Cricket Game (HTML, CSS, JavaScript)

A fun and interactive Cricket Game built using HTML, CSS, and JavaScript.  
This game simulates a simple cricket match where users can play, and the result is displayed after each round.

## 🚀 Features

- 🏏 Play cricket with simple controls
- 🎮 Interactive gameplay using JavaScript
- 📊 Displays result after each game
- 🔢 Score tracking system
- 🎨 Clean and responsive UI
- ⚡ Instant result updates (Win / Lose / Draw)

## 🛠 Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (Vanilla JS)** – Game logic & interactivity  

## 🎮 How to Play
- Click on the available game options (Bat / Bowl / etc.)
- The computer will randomly choose its move
- The result will be displayed:
    🟢 Win
    🔴 Lose
    🟡 Draw
- Score updates automatically after each round

## 📊 Game Logic
- Random choice is generated using JavaScript
- User input is compared with computer choice
- Result is calculated based on game rules
- Scoreboard updates dynamically
